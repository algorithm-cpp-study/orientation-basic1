# 기초 코드 작성 요령 1
컴퓨터는 1초에 대략 3-5억개 연산을 처리할 수 있다.  
예를 들어 시간 제한이 1초라고 하면, 당신의 프로그램은 3-5억 번의 연산 안에 답을 내고 종료되어야 한다는 의미이다.  
  
다음 함수는 5n + 3번의 연산을 수행한다. 즉, n에 비례한다(O(n))  
```c++
int func1(int arr[], int n) {
  int cnt = 0;
  for (int i = 0; i < n; i++) {
    if (arr[i] % 5 == 0) cnt++;
  }
  return cnt;
}
```
